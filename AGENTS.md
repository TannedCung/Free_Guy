# Cloud Agent Instructions

## Available skills

- `skills/cloud_agents_starter_skill.md`
  - Use this first when you need to run, test, or manually verify anything in this repo.
  - It includes practical setup, login, app startup, API smoke checks, and area-specific test workflows.

## Default workflow for cloud agents

1. Read `skills/cloud_agents_starter_skill.md`.
2. Follow the relevant section for the area you are changing (`frontend`, `frontend_server`, or `backend_server`).
3. Prefer targeted tests over full-suite runs.
4. When you learn a repeatable runbook trick, update the skill file in the same PR.
