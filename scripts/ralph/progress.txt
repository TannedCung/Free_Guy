## Codebase Patterns
- Access control for simulation endpoints: check `sim.owner == request.user` OR `SimulationMembership` active member OR `sim.visibility == PUBLIC`
- Pre-commit hooks run ruff + ruff-format; always re-stage after first failed commit attempt
- `SimulationStepCache` model: unique_together (simulation, step, stage); `data` is JSONField keyed by persona name
- `_can_observe` helper lives in `simulation_views.py` but api_views.py does its own inline check (same logic)
- URL patterns: new simulation endpoints go in `urls.py` between the SSE and step endpoints section
- ESLint rule `react-hooks/set-state-in-effect` blocks synchronous setState in useEffect body; wrap async fetches in useCallback and call them from within the effect instead
